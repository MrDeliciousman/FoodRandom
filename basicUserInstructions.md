# Basic user instructions
* 1.) The main fragment will take you automaticly to a edit text field and if not press on the edit text field indicated by "please enter ingredient"
* 2.) Here enter an ingredient 
* 3.) Press randomize to make a request to the api and recive results
* 4.) press back to get rid of the text editor
* 5.) Select the save button to save the response
* 6.) from here select the Saved Recipes tab from the bottom nav
* 7.) here select the saved recipe you want to view 
* 8.) Once selected  and finished viewing press the back arrow key to go back to the list of saved fragments to select something else
* 9.) If you wish to randomize another item simply press on the randomize button on the bottom nav   to go to the randomize fragment again and repeat steps 1-8.
